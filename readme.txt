Woocommerce for Price allows you to display call for price text, for woocommerce product.

== Description ==
Woocommerce for Price allows you to display call for price text, for woocommerce product.

= Features include: =

* Change Call for Price Text easy.
* Specific Product to use this feature

= How to Use: =
1. Make sure Woocommerce Call for Price Setting is enabled. Go to the Woocommerce Setting and Choose Call for Price Tab.
2. Edit the product you want to change to Call for Price, on On Woocommerce Product Setting enable Call for Price
3. Click Update

== Installation ==

1. Install Woocommerce Call for Price either via the WordPress.org plugin directory, or by uploading the files to your server
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the Woocommerce Call for Price settings and set your options.
